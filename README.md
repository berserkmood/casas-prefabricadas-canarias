# Casas Prefabricadas Canarias

Landing page para generación de leads cualificados del sector de casas prefabricadas en Tenerife y Gran Canaria.

## 🎯 Modelo de negocio
Conectamos particulares con constructoras verificadas. Los usuarios rellenan un formulario detallado y nosotros vendemos esos leads a las empresas.

## 🛠️ Tecnologías
- HTML5 semántico
- Tailwind CSS (vía CDN)
- Formspree para envío de formularios
- Diseño responsive

## 📁 Estructura
- `index.html` - Página principal
- `tenerife.html` - Landing específica Tenerife
- `gran-canaria.html` - Landing específica Gran Canaria
- `gracias.html` - Página post-envío
- `assets/` - CSS, JS, imágenes

## 🚀 Cómo probar localmente
1. Clona el repo
2. Abre cualquier archivo `.html` en tu navegador
3. No necesita build ni dependencias

## 📩 Formulario
Los datos se envían a Formspree y se redirige a `gracias.html`. Configurado para leads cualificados con 9 campos.
